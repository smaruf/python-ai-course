## üìñ Recommended Books

### Coding & Algorithms
1. **Cracking the Coding Interview** by Gayle Laakmann McDowell
   - Essential for interview prep
   - 189 programming questions
   - Detailed solutions

2. **Elements of Programming Interviews in Python** by Adnan Aziz
   - Python-focused
   - Advanced problem-solving techniques

3. **Introduction to Algorithms** by CLRS
   - Comprehensive algorithms reference
   - Academic depth

### Database
1. **Oracle Database 12c SQL** by Jason Price
   - Complete SQL reference
   - Oracle-specific features
   - Hands-on examples

2. **Expert Oracle Database Architecture** by Thomas Kyte
   - Deep dive into Oracle internals
   - Performance tuning
   - Best practices

3. **Pro Oracle SQL** by Karen Morton
   - Advanced SQL techniques
   - Query optimization
   - Execution plans

### System Design
1. **Designing Data-Intensive Applications** by Martin Kleppmann
   - Modern distributed systems
   - Database internals
   - Essential reading

2. **System Design Interview** by Alex Xu (Volumes 1 & 2)
   - Popular system design scenarios
   - Visual explanations
   - Step-by-step solutions

3. **Database Internals** by Alex Petrov
   - How databases work internally
   - Storage engines
   - Distributed databases

## üí° Tips for Using Resources

1. **Start with fundamentals** - Don't jump to advanced topics
2. **Practice consistently** - Daily practice beats weekend cramming
3. **Focus on understanding** - Don't just memorize solutions
4. **Use multiple resources** - Different explanations help understanding
5. **Join communities** - Learn from others' experiences
6. **Build projects** - Apply what you learn
7. **Track progress** - Use this repository to document learning
