## üíª AI Development Tools & Libraries

### Python Libraries for Oracle AI
```bash
# Oracle Database connectivity
pip install oracledb  # Modern python-oracledb driver

# Embedding models
pip install sentence-transformers  # Local embedding models
pip install openai  # OpenAI embeddings
pip install cohere  # Cohere embeddings

# RAG frameworks
pip install langchain  # LangChain for RAG
pip install llama-index  # LlamaIndex for RAG

# Vector operations
pip install numpy
pip install faiss-cpu  # For comparison/testing

# OCI SDK
pip install oci  # Oracle Cloud Infrastructure SDK
```
