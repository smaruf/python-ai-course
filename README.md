## üöÄ Quick Start Guide

### Prerequisites
```bash
# Core requirements
Python 3.8+
pip (Python package manager)
Git

# Optional (depending on project)
Docker
Node.js
Oracle Database 23ai/26ai
Go 1.19+
```

### Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/smaruf/python-ai-course.git
cd python-ai-course
```

2. **Choose a project** (see recommendations below)

3. **Install dependencies**
```bash
cd <project-directory>
pip install -r requirements.txt
```

4. **Follow the project-specific README** for detailed instructions
